# WhatsApp Bot 🤖

This is a simple WhatsApp bot built using Node.js.

## Features

* Auto reply messages
* Command system
* Easy to deploy
* Runs on Termux or PC

## Installation

1. Install Node.js
2. Clone this repository
3. Install dependencies

```
npm install
```

4. Start the bot

```
node bot.js
```

## Commands

* hi → bot replies hello
* help → shows command list
